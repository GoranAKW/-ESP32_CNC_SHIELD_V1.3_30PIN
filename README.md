New version of  ESP32 CNC SHIELD V1.3 30PIN

    6 Pinsockets 8x1
    
    3 JST Feamale 4x1
    
    7 JST Feamale 2x1
    
    So you can build your own cables
    
    3 JST Male 4x1
    
    7 JST Male 2x1
    
    30 Crimp terminals
